# STGCCA
This folder contains the code to reproduce the results in the paper "Sparse Tensor Generalized Canonical Correlation Analysis" 

Here is a brief description of the sub directories present in this folder:

## Prerequisites

RGCCA framework, stgcca branch available here: https://github.com/rgcca-factory/RGCCA

### STGCCA-2D_simulations folder

This folder contains the code to reproduce the results of the simulations using 2D shapes as canonical vectors. 

### STGCCA-chemical_mixtures

This folder contains the code to reproduce the results on the chemical mixture dataset.
